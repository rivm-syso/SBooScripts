#### Erosion

In R processes across the same dimension, such as Runoff and Erosion are
defined seperately. The engine adds the rate constants (k’s)
automatically for the K matrix. This approach is in favor because of the
clearly defined process functions.

The factor CORRrunoff (Correction factor depth dependent soil
concentration) however is calculated as EXP((-1/0.1) \* 0) \* (1/0.1) \*
DEPTH / (1-EXP((-1/0.1) \* DEPTH)) for the assumed pentration depth of
0.1 m. To make sure this remains the same for both Runoff and erosion,
we define a variable to calculate it. The only parameter, so far, is
DEPTH, which has been renamed to vertDistance in the R version (combined
with height).

At Regional and Continental scale Erossion and Runoff all goes to
freshwater (w0 an w1), whereas at global scales (Tropic, Moderate and
Arctic) all Erosion and Runoff goes to sea (w2).

At Regional and Continental scale the fraction of Erossion and Runoff
going to either w0 or w1 is calculated using a variable: FracROWatComp.

#### Runoff Molecular

Runoff is implemented in a similar way. Here the formula is: RunoffFlow
/ Kscompw \* CORRrunoff / Volume \* FracROWatComp. For organic compounds
we need Kscompw (Ksw), which is described in section [4.0
partitioning](#0), as only the fraction in water can move with Runoff.
The correction factor is explained above. The RunoffFlow from soil is
needed, which is dependant on the Area, FRACrun and the RAINrate. And at
last the fraction of Runoff can go to either the upstream (w0) or
downstream (w1) freshwater compartment as calculated by v_FracROWatComp.
FracROWatComp is now calculated based on the surface fraction (landFRAC)
of w0 versus w1. This can or should be adjusted when more specific data
is available on the land surface area contributing to the upstream (w0)
versus downstream (w1) Runoff.

#### Runoff Particulate species

For particulate species Runoff is not appliable for very large
particles, meaning that `VegInterceptFrac = 1`. These particles behave
more like other particulates that make up the soil or ar at least
intercepted by vegetation reducing the Runoff rate. As of version
2025.10.0 a size cutoff, SizeRunoff, is introduced above which Runoff is
either set to 0 or to a value reduced by the `VegInterceptFrac` (see
SubCompartSheet.csv) parameter as calculated in the `IntrcptFrac`
variable. See example code below. `SizeRunoff` is set at the scale
level, where currently the `SizeRunoff` is set at 100 micrometer for all
scales. This will particularly influence the runoff of microplastics, as
the primary particle can already have a diameter larger than 100
micrometers, so this size cutoff for Runoff can affect all species in
such a case, where for nanomaterials (\< 100 nm) only the largest
species is affected.
